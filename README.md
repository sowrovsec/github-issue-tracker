## Question Answer :

1️⃣ What is the difference between var, let, and const?

Ans: The difference between var, let, and const is that they are all used for declaring variables in JS. But the var is kind of an old way to declare a variable in js and it is function-scoped. It can also be redecleared or updated. let is a modern way to declare a variable in js and it is block-scoped. And const is also block-scoped, but the value can't be changed after assigned.

2️⃣ What is the spread operator (...)?

Ans: Spread Operator can be used to copy all elements of an array into a new array or copies all object properties into a new object for accessing those easily.


3️⃣ What is the difference between map(), filter(), and forEach()?
Ans:  The difference between map(), filter(), and forEach().  map() loops through an array and retun new array. filter() also goes through an array and returns a new array with fulfilling the condition. forEach() also loops through an array and iterates for each element, but does not return any new array.
